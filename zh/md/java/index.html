            # JavaSE知识体系

            JavaSE 是 Java Platform, Standard Edition 的缩写，是 Java 平台中最基础、最核心的部分。它提供了 Java 编程语言的基础类库、Java 虚拟机（JVM）和 Java 开发工具包（JDK）等组成部分。

            JavaSE 中的基础类库包括了 Java 编程语言的核心类库和辅助类库，如集合框架、输入/输出、网络通信、多线程、反射、XML解析、安全和加密等。JavaSE 还提供了 Java 虚拟机（JVM），它是一种能够在不同操作系统上运行 Java 程序的虚拟机，可以将 Java 代码转换成机器指令。

            本部分主要对除 JVM 外的知识点进行梳理归纳。
