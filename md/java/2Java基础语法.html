                                                # Java基础语法

                                                # 1.注释

                                                Java 注释是在 Java 程序中用于解释、描述代码和文档的一种方式。Java 注释是以双斜线（//）或者是在 /**/ 中间的注释。Java 注释不会被编译器编译，因此在程序运行时不会影响程序的执行。

                                                Java 注释主要有以下几种：

                                                1. 单行注释：以双斜线（//）开头，注释内容从双斜线开始一直到行末。

                                                2. 多行注释：以“/*”开头，“*/”结尾，注释内容可以跨越多行。

                                                3. 文档注释：以“/**”开头，“*/”结尾，一般用于描述类、方法、变量等的作用和用法，可以通过 Javadoc 工具生成文档。

                                                  关于如何使用 Javadoc 工具生成文档请参见：如何使用 Javadoc 工具生成文档open in new window

                                                下面是 Java 注释的示例：

                                                // 这是单行注释

/*
 * 这是多行注释
 * 注释内容可以跨越多行
 */

/**
 * 这是文档注释，用于描述类、方法、变量等的作用和用法
 * @author smonk
 */
public class MyClass {
    // ...
}

                                                下面是一些文档注释常用注解：

                                                • @author 标明开发该模块的作者，多个作者之间使用英文逗号 , 分割

                                                • @version 标明该类模块的版本

                                                • @see 参考转向，也就是相关主题

                                                • @since 从哪个版本开始增加的

                                                • @param 对方法中某参数的说明，如果没有参数就不能写

                                                  格式：@param 形参名 形参类型 形参说明

                                                • @return 对方法返回值的说明，如果方法的返回值类型是 void 就不能写

                                                  格式：@return 返回值类型 返回值说明

                                                • @throws/@exception 对方法可能抛出的异常进行说明 ，如果方法没有用 throws 显式抛出的异常就不能写

                                                  格式：@exception 异常类型 异常说明

                                                # 2.标识符

                                                在 Java 中，标识符是用于表示变量、方法、类、接口、包或其他用户自定义项的名称。标识符的命名规则如下：

                                                1. 标识符必须以字母、美元符号（$）或下划线（_）开头。
                                                2. 标识符可以包含字母、数字、美元符号或下划线。
                                                3. 标识符的长度没有限制。
                                                4. 标识符是大小写敏感的。
                                                5. 不能使用 Java 的关键字作为标识符。

                                                以下是一些有效的标识符示例：

                                                count
sum_total
myVar
MyClass
$price

                                                以下是一些无效的标识符示例：

                                                1stNumber   //不能以数字开头
total%     //不能使用特殊字符
class      //关键字不能用作标识符

                                                上面的是一些硬性规则，当然还有一些不得不遵守的软性规则：

                                                1. 命名要求见名知意
                                                2. 类名、接口名等每个单词的首字母都大写，形式：XxxYyyZzz
                                                3. 变量、方法名等从第二个单词开始首字母大写，其余字母小写，形式：xxxYyyZzz
                                                4. 包名等每一个单词都小写，单词之间使用点 . 分割，形式：xxx.yyy.zzz
                                                5. 常量名等每一个单词都大写，单词之间使用下划线 _ 分割，形式：XXX_YYY_ZZZ

                                                # 4.数据类型

                                                # 4.1 基本数据类型

                                                Java 中的基本数据类型是用于存储基本类型数据的预定义数据类型。Java 的基本数据类型可以分为以下几种类型：

                                                1. 整数类型（4 种）：

                                                  类型字节取值范围
                                                  byte1-128 到 127
                                                  short2-32768 到 32767
                                                  int4-2147483648 到 2147483647
                                                  long8-9223372036854775808 到 9223372036854775807

                                                2. 浮点类型（2 种）：

                                                  类型字节取值范围精度
                                                  float（单精度浮点数）41.4E-45 到 3.4028235E386-7 位小数
                                                  double（双精度浮点数）84.9E-324 到 1.7976931348623157E30815-16 位小数

                                                3. 布尔类型（1 种）：

                                                  类型描述取值
                                                  boolean表示真或假true 和 false

                                                4. 字符类型（1 种）：

                                                  类型字节字符集取值范围
                                                  char2Unicode 字符\u0000 到 \uffff

                                                以下是使用基本数据类型声明变量的示例：

                                                byte age = 20;
short year = 2022;
int count = 100000;
long balance = 1000000000L; //必须在后面加上 L （大小写皆可），否则默认为 int 类型
float salary = 5000.50f; // 必须在后面加上 f 或 F，否则默认为 double 类型
double taxRate = 0.15;
boolean isMarried = true;
char gender = 'M'; // 字符使用单引号

                                                # 4.2 引用数据类型

                                                Java 中的引用数据类型是用于存储对象引用的数据类型。Java 中所有的类和接口都是引用数据类型。引用数据类型的值指向对象的内存地址，而不是实际的值。

                                                Java 中常用的引用数据类型包括：

                                                1. 类（Class）：类是一种引用数据类型，用于定义对象的属性和方法。在 Java 中，所有的类都继承自 Object 类。
                                                2. 接口（Interface）：接口是一种引用数据类型，用于定义方法的规范，但不包含方法的实现。在 Java 中，一个类可以实现一个或多个接口。
                                                3. 数组（Array）：数组是一种引用数据类型，用于存储同一类型的多个元素。在 Java 中，数组可以是一维的、二维的或多维的。
                                                4. 枚举（Enum）：枚举是一种引用数据类型，用于定义一组具有相同类型的常量。在 Java 中，枚举类型是一种特殊的类。
                                                5. 字符串（String）：字符串是一种引用数据类型，用于存储文本字符串。在 Java 中，字符串是不可变的对象。

                                                以下是使用引用数据类型声明变量的示例：

                                                MyClass obj = new MyClass(); // 类
MyInterface myIntf = new MyInterfaceImpl(); // 接口
int[] nums = {1, 2, 3, 4}; // 数组
Color color = Color.RED; // 枚举
String name = "Alice"; // 字符串，使用双引号

                                                # 5.变量

                                                在 Java 中，变量是用于存储数据值（可以改变）的占位符。Java 中的每个变量都具有一个类型（可以是基本类型与引用类型），用于指定变量可以存储的数据类型。变量在使用之前必须先声明，并且可以在声明时进行初始化。

                                                Java 种变量的三要素：

                                                • 数据类型：变量的数据类型决定了在内存中开辟多大空间，也决定了该变量可以存什么值。
                                                • 变量名：见名知意非常重要。
                                                • 变量值：基本数据类型的变量存储数据值，引用数据类型的变量存储地址值。

                                                Java 中的变量可以分为以下几种类型：

                                                1. 局部变量（Local Variables）：局部变量声明在方法、构造函数或代码块中，并且在这些方法、构造函数或代码块中可见。局部变量必须在使用之前声明和初始化。如果没有初始化，编译器会报错。
                                                2. 成员变量（Instance Variables）：成员变量声明在类中，但在方法、构造函数或代码块之外。成员变量的值将在对象创建时分配，并且在对象销毁时销毁。成员变量可以是私有的、公共的或受保护的。
                                                3. 静态变量（Static Variables）：静态变量声明在类中，但在方法、构造函数或代码块之外，并且使用 static 关键字声明。静态变量的值在类加载时分配，并且在类卸载时销毁。

                                                以下是使用变量的示例：

                                                public class MyClass {
    int instanceVar; // 成员变量
    static int staticVar; // 静态变量

    public void myMethod() {
        int localVar = 10; // 局部变量
        System.out.println(localVar); // 输出 10
        System.out.println(instanceVar); // 输出 0，因为成员变量没有初始化
        System.out.println(staticVar); // 输出 0，因为静态变量没有初始化
    }
}

                                                在上面的示例中，instanceVar 是成员变量，staticVar 是静态变量，localVar 是局部变量。在 myMethod() 方法中，我们可以访问这三个变量。

                                                下面是一些使用注意事项：

                                                • 给变量赋的值类型必须与变量声明的类型一致或兼容（<=）
                                                • 可以使用其他变量或者表达式给变量赋值
                                                • 变量的第一次赋值称为初始化，变量的再赋值称为修改变量的值。
                                                • 变量需要先声明后使用，否则会报“找不到符号”错误。
                                                • 变量在使用之前必须初始化，否则会报“未初始化”错误。
                                                • 变量有一定的作用域，如果超过作用域，也会报“找不到符号”错误。
                                                • 变量在同一个作用域中不能重名。

                                                # 6.字面量

                                                在 Java 中，字面量是程序中直接使用的常量值，它们可以是整数、浮点数、布尔值、字符、字符串或空值。

                                                下面是 Java 中各种类型的字面量的示例：

                                                1. 整数字面量：整数字面量是指整数值，可以是十进制、八进制或十六进制。例如：

                                                  int num1 = 10;       // 十进制整数
int num2 = 012;      // 八进制整数，等价于十进制的 10
int num3 = 0xA;      // 十六进制整数，等价于十进制的 10

                                                2. 浮点数字面量：浮点数字面量是指浮点数值，可以是单精度浮点数（float 类型）或双精度浮点数（double 类型）。例如：

                                                  float num1 = 3.14f;      // 单精度浮点数
double num2 = 3.14;      // 双精度浮点数

                                                3. 布尔字面量：布尔字面量是指布尔值，可以是 true 或 false。例如：

                                                  boolean flag1 = true;    // 布尔值为 true
boolean flag2 = false;   // 布尔值为 false

                                                4. 字符字面量：字符字面量是指单个字符，使用单引号括起来。例如：

                                                  char ch1 = 'A';          // 字符 A
char ch2 = '\n';         // 换行符
char ch3 = '\u0041';     // 字符 A 的 Unicode 编码

                                                5. 字符串字面量：字符串字面量是指一个或多个字符组成的字符串，使用双引号括起来。例如：

                                                  String str1 = "Hello, world!";     // 字符串字面量
String str2 = "";             // 空字符串

                                                6. 空字面量：空字面量表示 null，表示没有值。例如：

                                                  String str = null;      // 空字符串

                                                字面量在 Java 中常常被使用，可以用于初始化变量、进行比较和作为方法参数等。

                                                # 7.常量

                                                在 Java 中，常量是指在程序中固定不变的值。Java 中的常量可以分为字面值常量和符号常量两种类型。

                                                1. 字面值常量：字面值常量是程序中直接使用的值，例如整数、浮点数、布尔值和字符。例如，下面的代码定义了一个整数常量：

                                                  int num = 10;

                                                2. 符号常量：符号常量是指使用常量名称代替常量值的值。在 Java 中，符号常量通常使用 final 关键字进行定义。例如，下面的代码定义了一个符号常量：

                                                  final double PI = 3.14159;

                                                  在上面的代码中，PI 是一个符号常量，它的值为 3.14159。使用 final 关键字定义的常量是不可修改的，因此我们可以在程序中引用该常量，而不用担心其值被修改。

                                                3. 枚举常量：枚举常量是一组具有相同类型的常量。在 Java 中，枚举类型是一种特殊的类，用于定义一组常量。例如，下面的代码定义了一个枚举类型：

                                                  enum Color {
    RED, GREEN, BLUE
}

                                                  在上面的代码中，Color 是一个枚举类型，它包含了三个常量：RED、GREEN 和 BLUE。

                                                在 Java 中，常量通常使用大写字母表示，这有助于使常量在程序中更易于识别。例如，我们可以将上面的 PI 常量改写为：

                                                final double PI = 3.14159;

                                                在程序中使用常量有许多好处，例如使代码更容易阅读和维护，减少编码错误，并提高代码的可重用性。

                                                # 8.数据类型转换

                                                # 8.1 基本数据类型转换

                                                Java 中的基本数据类型转换指的是将一个数据类型的值转换为另一个数据类型的值。Java 中的数据类型转换分为两种：自动类型转换和强制类型转换。（boolean类型不参与）

                                                1. 自动类型转换：当两种数据类型兼容时，Java 会自动将较小的数据类型转换为较大的数据类型，这种转换称为自动类型转换。例如：

                                                  int num1 = 100;
double num2 = num1;    // 自动将 int 类型转换为 double 类型

                                                  在上面的例子中，变量 num1 是 int 类型，变量 num2 是 double 类型，Java 会自动将 num1 转换为 double 类型，然后将结果赋值给 num2。

                                                  基本数据类型的转换规则如图所示：

                                                  当存储范围小的数据类型与存储范围大的数据类型一起混合运算时，会按照其中最大的类型运算。示例代码如下：

                                                  int num1 = 1;
byte num2 = 1;
double num3 = 1.0;

double sum = num1 + num2 + num3; //混合运算，升级为 double

                                                  当 byte,short,char 数据类型进行算术运算时，按照 int 类型处理。示例代码如下：

                                                  byte b1 = 1;
byte b2 = 2;
byte b3 = b1 + b2; //编译报错，b1 + b2 自动升级为 int

char c1 = '0';
char c2 = 'A';
System.out.println(c1 + c2); //113

                                                2. 强制类型转换：当两种数据类型不兼容时，或者需要将较大的数据类型转换为较小的数据类型时，可以使用强制类型转换。强制类型转换使用括号将要转换的数据类型括起来，放在需要转换的变量或常量前面。例如：

                                                  double num1 = 3.14;
int num2 = (int)num1;    // 强制将 double 类型转换为 int 类型

                                                  在上面的例子中，变量 num1 是 double 类型，变量 num2 是 int 类型，我们使用强制类型转换将 num1 转换为 int 类型，然后将结果赋值给 num2。

                                                  需要注意的是，进行强制类型转换时可能会丢失精度或者出现数据溢出，因此应该谨慎使用。

                                                  示例代码：

                                                  int num1 = (int)3.14; //强制类型转换，损失精度

double num2 = 1.2;
int num3 = (int)num2; //损失精度

int num4 = 200;
byte num5 = (byte)num4; //溢出，超出 byte 取值范围

                                                # 8.2 引用类型数据类型转换

                                                在 Java 中，引用数据类型的转换可以分为向上转型和向下转型两种。

                                                1. 向上转型：将一个子类类型的引用转换为父类类型的引用。向上转型不需要进行强制类型转换，因为子类对象是一种特殊的父类对象。例如：

                                                  // 父类
class Animal {
  public void eat() {
    System.out.println("Animal eat");
  }
}

// 子类
class Cat extends Animal {
  public void eat() {
    System.out.println("Cat eat");
  }
}

public class Test {
  public static void main(String[] args) {
    Cat cat = new Cat();
    Animal animal = cat;  // 向上转型（子类类型的引用转换为父类类型的引用）
    animal.eat();    // 输出 "Cat eat"
  }
}

                                                  在上面的例子中，Cat 是 Animal 的子类，我们将 Cat 类型的对象 cat 赋值给 Animal 类型的变量 animal，这就是向上转型。由于子类对象也是一种特殊的父类对象，所以我们可以通过 animal 调用 eat() 方法，输出的结果是 "Cat eat"。

                                                2. 向下转型：将一个父类类型的引用转换为子类类型的引用。向下转型需要进行强制类型转换，因为父类对象不一定是子类对象，如果强制类型转换失败，就会抛出 ClassCastException 异常。例如：

                                                  // 父类
class Animal {
  public void eat() {
    System.out.println("Animal eat");
  }
}

// 子类
class Cat extends Animal {
  public void eat() {
    System.out.println("Cat eat");
  }

  public void meow() {
    System.out.println("Meow");
  }
}

public class Test {
  public static void main(String[] args) {
    Animal animal = new Animal();
    Cat cat = (Cat)animal; // 向下转型（父类类型的引用转换为子类类型的引用）
    cat.meow();   // 抛出 ClassCastException 异常
  }
}

                                                  在上面的例子中，我们先创建了一个 Animal 类型的对象 animal，然后将其赋值给 Cat 类型的变量 cat，这就是向下转型。由于 animal 实际上并不是 Cat 类型的对象，所以在进行强制类型转换时会抛出 ClassCastException 异常。

                                                需要注意的是，向上转型是安全的，因为子类对象是一种特殊的父类对象，向下转型是不安全的，因为父类对象不一定是子类对象，需要进行类型检查和转换。在进行向下转型时，应该先使用 instanceof 运算符进行类型检查，以确保类型转换的安全性。

                                                示例代码：

                                                public static void main(String[] args) {
    Animal animal = new Cat();
    if(animal instanceof Cat){
       Cat cat = (Cat)animal; // 向下转型
    	cat.meow();  
    }  
  }

                                                # 9.运算符

                                                # 9.1 算数运算符

                                                Java 算数运算符用于进行数值计算。以下是 Java 中的算术运算符：

                                                运算符描述
                                                +加法运算符，用于相加两个数或者连接两个字符串
                                                -减法运算符，用于减去一个数或者取反一个数
                                                *乘法运算符，用于相乘两个数
                                                /除法运算符，用于相除两个数
                                                %取模运算符，用于取两个数相除后的余数
                                                ++自增运算符，将变量的值增加 1，有前缀和后缀两种形式
                                                --自减运算符，将变量的值减少 1，有前缀和后缀两种形式

                                                以下是算术运算符的示例：

                                                int a = 10, b = 5;
int c = a + b; // c 的值为 15
int d = a - b; // d 的值为 5
int e = a * b; // e 的值为 50
int f = a / b; // f 的值为 2
int g = a % b; // g 的值为 0

// 余数的符号只看被除数
System.out.println(5%2);	//1
System.out.println(5%-2);	//1	(-2)*(-2)+1 = 5
System.out.println(-5%2);	//-1  (-2)*2+(-1) = -4-1=-5
System.out.println(-5%-2);	//-1	

int i = 1;
i++; // i 的值为 2（先用后加）
int j = 1;
++j; // j 的值为 2（先加后用）

int k = 10;
int m = k++; // k 的值为 11，m 的值为 10（先用（10）后加（11））
int n = 10;
int p = ++n; // n 的值为 11，p 的值为 11（先加（11）后用（11））

                                                需要注意的是，在进行整数除法时，如果被除数和除数都是整数类型，则结果将取整。如果想得到精确的浮点数结果，需要将其中一个操作数转换为浮点数类型。另外，在取模运算中，余数的符号和被除数的符号一致。

                                                # 9.2 关系运算符

                                                Java 中的关系运算符用于比较两个操作数之间的大小、相等性等关系，返回布尔类型的值（true 或 false）。

                                                以下是 Java 中的关系运算符：

                                                运算符描述
                                                ==判断两个操作数是否相等，如果相等则返回 true，否则返回 false
                                                !=判断两个操作数是否不相等，如果不相等则返回 true，否则返回 false
                                                >判断左操作数是否大于右操作数，如果是则返回 true，否则返回 false
                                                <判断左操作数是否小于右操作数，如果是则返回 true，否则返回 false
                                                >=判断左操作数是否大于等于右操作数，如果是则返回 true，否则返回 false
                                                <=判断左操作数是否小于等于右操作数，如果是则返回 true，否则返回 false

                                                以下是关系运算符的示例：

                                                int a = 10, b = 5;
boolean c = (a == b); // c 的值为 false
boolean d = (a != b); // d 的值为 true
boolean e = (a > b); // e 的值为 true
boolean f = (a < b); // f 的值为 false
boolean g = (a >= b); // g 的值为 true
boolean h = (a <= b); // h 的值为 false

                                                需要注意的是:

                                                • 关系运算符的操作数必须是数值类型或者可以转换为数值类型的类型。
                                                • 对于引用类型，关系运算符比较的是两个对象的引用地址，而不是对象的内容。

                                                # 9.3 逻辑运算符

                                                Java 中的逻辑运算符用于对布尔类型的操作数进行逻辑运算，返回布尔类型的结果。

                                                以下是 Java 中的逻辑运算符：

                                                运算符描述
                                                &&逻辑与运算符，如果两个操作数都为 true，则返回 true，否则返回 false
                                                ||逻辑或运算符，如果两个操作数中有一个为 true，则返回 true，否则返回 false
                                                !逻辑非运算符，如果操作数为 true，则返回 false，否则返回 true

                                                以下是逻辑运算符的示例：

                                                boolean a = true, b = false;
boolean c = (a && b); // c 的值为 false
boolean d = (a || b); // d 的值为 true
boolean e = !a;       // e 的值为 false

                                                需要注意的是，在进行逻辑运算时，Java 中的短路特性是非常重要的，即如果第一个操作数已经能够确定整个表达式的值，则不会对第二个操作数进行求值。例如，如果使用逻辑与运算符（&&）时，第一个操作数为 false，则不会对第二个操作数进行求值，因为整个表达式的结果已经可以确定为 false。

                                                # 9.4 位运算符

                                                Java 中的位运算符用于对二进制位进行操作，是对整数类型的操作。

                                                以下是 Java 中的位运算符：

                                                运算符描述
                                                &按位与运算符，两个操作数的二进制位都为 1 时，结果才为 1，否则为 0
                                                |按位或运算符，两个操作数的二进制位中有一个为 1 时，结果就为 1，否则为 0
                                                ^按位异或运算符，两个操作数的二进制位不相同时，结果为 1，否则为 0
                                                ~按位取反运算符，操作数的每个二进制位取反，即 0 变成 1，1 变成 0
                                                <<左移运算符，将操作数的二进制位左移指定的位数，低位补 0
                                                >>右移运算符，将操作数的二进制位右移指定的位数，高位补符号位
                                                >>>无符号右移运算符，将操作数的二进制位右移指定的位数，高位补 0

                                                注意

                                                1. 位运算符的运算过程都是基于补码运算，但是看结果得换成原码，然后再换成十进制看结果。（相关阅读：一篇文章学会数制open in new window、计算机如何存储数据）
                                                2. byte、short、char在计算时按照 int 类型处理
                                                3. 在进行位运算时，位运算符的操作数必须是整数类型或者可以转换为整数类型的类型。

                                                • 按位与运算符

                                                  9&7 = 1

                                                  -9&7 = 7

                                                • 按位或运算符

                                                  9|7 = 15

                                                  -9|7 = -9

                                                • 按位异或运算符

                                                  9^7 = 14

                                                  -9^7 = -16

                                                • 按位取反运算符

                                                  ~9 = -10

                                                  ~-9 = 8

                                                • 左移运算符

                                                  左移几位就相当于乘以2的几次方。当左移的位数n超过该数据类型的总位数时，相当于左移（n-总位数）位。

                                                  3<<4 类似于 3*2 的 4 次 = 3*16 = 48

                                                  -3<<4 类似于 -32的4次= -316 = -48

                                                • 右移运算符

                                                  类似于除以2的n次，如果不能整除，向下取整。

                                                  69>>4 类似于 69/2的4次 = 69/16 =4

                                                  -69>>4 类似于 -69/2的4次 = -69/16 = -5

                                                • 无符号右移运算符

                                                  往右移动后，左边空出来的位直接补0，不看符号位。

                                                  正数：和右移一样

                                                  负数：右边移出去几位，左边补几个0，结果变为正数

                                                  69>>>4 类似于 69/2的4次 = 69/16 =4

                                                  -69>>>4 结果：268435451

                                                # 9.5 赋值运算符

                                                Java 中的赋值运算符用于将一个值赋给一个变量或表达式，并返回赋值后的值。

                                                以下是 Java 中的赋值运算符：

                                                运算符描述
                                                =简单的赋值运算符，将一个值赋给一个变量或表达式
                                                +=加法赋值运算符，相当于 a = a + b
                                                -=减法赋值运算符，相当于 a = a - b
                                                *=乘法赋值运算符，相当于 a = a * b
                                                /=除法赋值运算符，相当于 a = a / b
                                                %=取模赋值运算符，相当于 a = a % b
                                                <<=左移赋值运算符，相当于 a = a << b
                                                >>=右移赋值运算符，相当于 a = a >> b
                                                >>>=无符号右移赋值运算符，相当于 a = a >>> b
                                                &=按位与赋值运算符，相当于 a = a & b
                                                |=按位或赋值运算符，相当于 a = a | b
                                                ^=按位异或赋值运算符，相当于 a = a ^ b

                                                以下是赋值运算符的示例：

                                                int a = 10, b = 5;
a += b;   // 相当于 a = a + b，a 的值为 15
a -= b;   // 相当于 a = a - b，a 的值为 10
a *= b;   // 相当于 a = a * b，a 的值为 50
a /= b;   // 相当于 a = a / b，a 的值为 2
a %= b;   // 相当于 a = a % b，a 的值为 0
a <<= 2;  // 相当于 a = a << 2，a 的值为 40
a >>= 2;  // 相当于 a = a >> 2，a 的值为 10
a >>>= 2; // 相当于 a = a >>> 2，a 的值为 2
a &= b;   // 相当于 a = a & b，a 的值为 0
a |= b;   // 相当于 a = a | b，a 的值为 5
a ^= b;   // 相当于 a = a ^ b，a 的值为 0

                                                需要注意的是，赋值运算符的优先级比大多数其他运算符都低，因此在使用赋值运算符时，应当注意运算的优先级，以免出现错误的结果。

                                                # 9.6 条件运算符

                                                Java 中的条件运算符（也称为三元运算符）是一个特殊的运算符，它需要三个操作数，通常用于简单的条件判断。其语法如下：

                                                条件 ? 表达式1 : 表达式2

                                                其中，条件是一个布尔表达式，如果为 true，则返回表达式1的值，否则返回表达式2的值。

                                                以下是条件运算符的示例：

                                                int a = 10, b = 5;
int max = (a > b) ? a : b;  // 如果 a 大于 b，则将 a 的值赋给 max，否则将 b 的值赋给 max
System.out.println("max = " + max);  // 输出结果为 "max = 10"

                                                在上面的示例中，如果 a 大于 b，则将 a 的值赋给 max，否则将 b 的值赋给 max。由于 a 的值为 10，大于 b 的值 5，因此 max 的值为 10。

                                                # 9.7 instanceof运算符

                                                在 Java 中，instanceof 运算符用于检查一个对象是否属于某个特定的类或其子类的实例。其语法如下：

                                                object instanceof class

                                                其中，object 是要检查的对象，class 是要检查的类或接口。

                                                如果 object 是 class 的实例或 class 的子类的实例，则 instanceof 运算符返回 true，否则返回 false。

                                                以下是 instanceof 运算符的示例：

                                                class Animal {}
class Dog extends Animal {}

public class Main {
  public static void main(String[] args) {
    Animal animal = new Dog();
    System.out.println(animal instanceof Animal); // 输出 true
    System.out.println(animal instanceof Dog);    // 输出 true
    System.out.println(animal instanceof Object); // 输出 true
  }
}

                                                在上面的示例中，Dog 类是 Animal 类的子类。首先，我们创建了一个 Dog 类的对象，并将其赋给一个 Animal 类型的变量 animal。然后，我们使用 instanceof 运算符检查 animal 是否是 Animal 类的实例、Dog 类的实例以及 Object 类的实例。由于 animal 是 Dog 类的实例，它也是 Animal 类的实例和 Object 类的实例，因此 instanceof 运算符返回 true。

                                                # 9.8 ++和--运算符

                                                在 Java 中，++ 和 -- 运算符分别用于将变量的值加一和减一。它们可以分别用作前缀和后缀运算符。

                                                • 如果它们用作前缀运算符，则它们会先对变量进行加一或减一，然后返回结果；
                                                • 如果它们用作后缀运算符，则它们会先返回变量的值，然后对变量进行加一或减一。

                                                以下是 ++ 和 -- 运算符的示例：

                                                int a = 10;
System.out.println(++a); // 输出 11
System.out.println(a);   // 输出 11

int b = 10;
System.out.println(b++); // 输出 10
System.out.println(b);   // 输出 11

int c = 10;
System.out.println(--c); // 输出 9
System.out.println(c);   // 输出 9

int d = 10;
System.out.println(d--); // 输出 10
System.out.println(d);   // 输出 9

                                                # 9.9 点号运算符

                                                在 Java 中，点号运算符 . 用于访问对象的属性和方法。它通常用于访问类的静态成员和实例成员。对于实例成员，我们需要先创建一个对象，然后使用对象名称和点号运算符来访问它们。对于静态成员，我们可以直接使用类名和点号运算符来访问它们。

                                                以下是点号运算符的一些示例：

                                                // 定义一个名为 person 的类
public class Person {
    String name;
    int age;
    public void sayHello() {
        System.out.println("Hello, my name is " + name);
    }
}

// 创建一个 Person 对象并访问其属性和方法
Person p = new Person();
p.name = "Alice";
p.age = 25;
p.sayHello();

// 访问类的静态成员
System.out.println(Math.PI);
System.out.println(Integer.MAX_VALUE);

                                                # 9.10 数组下标运算符

                                                在 Java 中，数组下标运算符 [] 用于访问数组中的元素。数组是一种特殊类型的对象，它可以存储一组相同类型的元素。数组中的每个元素都有一个唯一的索引，可以使用数组下标运算符来访问该元素。

                                                数组下标从 0 开始，因此第一个元素的索引是 0，第二个元素的索引是 1，以此类推。使用数组下标运算符时，需要在数组名称后面加上方括号，括号内放置要访问的元素的索引。

                                                以下是数组下标运算符的一些示例：

                                                // 定义一个长度为 5 的整型数组
int[] numbers = {1, 2, 3, 4, 5};

// 访问数组中的元素
int first = numbers[0];
int third = numbers[2];
numbers[4] = 10;

// 循环遍历数组并输出所有元素
for (int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}
